Capstone Project

Utilizing popular Python libraries and historical weather data for the Worcester MA region gathered from NOAA, I created a machine learning application that forecasts weather for either the next day or future days based on the current or latest available day's weather.

This model uses linear regression to make predictions. The use of Long Short-Term Memory (LSTM) via Tensorflow was considered and modeled, but a simpler linear regression model was chosen.

Below are the tools and libraries used:

  • Python
  • Visual Studio Code
  • Scikit-learn - data modeling and predictions
  • Pandas - data manipulation and analysis
  • NumPy - data manipulation and regression line fitting
  • Matplotlib - data visualzation
  • TKinter - application UI
