TravelBin - Travel Planner

Inspired by Wanderlog, Google Sheets, Pastebin, Partiful, and soon Splitwise.

TravelBin is a travel planning web application where you can collaboratively work with friends and family to plan your next trip. You can create an account, make a list for wherever you want to go, and start adding activities. Share the link to your unique trip to whoever you'd like and add them via email address if they have an account to co-edit your next vacation.

The idea for this came from other apps that do something similar but I had gripes about using. While Wanderlog is a great app for trip planning, I didn't like how "busy" the app looked. Features are aplenty but it became visually cluttered, moreso on the mobile app. This lead me to use the idea of Wanderlog, simplicity of PasteBin and Partiful's UI, and collaborative editing of Google Sheets to create my own app.

Below are the tools and libraries used:

  • Python
  • HTML, CSS, JavaScript
  • Visual Studio Code
  • Django - backend framework
  • Django Rest Framework - REST API building
  • SQLite - database
  • Google OAuth 2.0 - optional user account creation and authentication
  • React.js - frontend framework
  • Figma - design and planning
  • Jira - task and ticket management
