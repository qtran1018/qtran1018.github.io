Package Delivery Router - Traveling Salesperson Problem

Link to repository

WGU Data Structures and Algorithms Project

This command-line application simulates package delivery logistics in the Salt Lake City, UT area. It uses a nearest neighbor algorithm to generate efficient delivery routes for three trucks, each pre-loaded with packages. The system tracks and updates delivery times and statuses throughout the simulation.

Package data is provided via a spreadsheet, and a distance matrix defines the travel distances between all addresses. To prioritize speed and scalability, the implementation leverages hash maps for quick lookups and a nearest neighbor approach for route calculation—balancing simplicity and performance for real-world-like logistics handling.

Below are the tools and methods used:

  • Python
  • Visual Studio Code
  • Hashmap data structure
  • Nearest Neighbor Algorithm
