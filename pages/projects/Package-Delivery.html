Package Delivery Router - Traveling Salesperson Problem

Link to repository

WGU Data Structures and Algorithms Project

Using a nearest neighbor algorithm, this CLI-application calculates a delivery route for 3 trucks pre-loaded with packages going out in the Salt Lake City UT area. This simulates the delivery aspect and handles the updating of delivery time and status.

By choosing hash maps and the nearest neighbor algorithm, ease of implementation and speed was priorotized, allowing for effortless scaling while maintaining efficiency of the calculations.

Below are the tools and methods used:

  • Python
  • Visual Studio Code
  • Hashmap data structure
  • Nearest Neighbor Algorithm
